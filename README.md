# Project sprint7

Этот проект использует rest-assured на джаве для выполнения авто-тестов

Используемые технологии:
- allure-maven
- aspectjweaver
- gson

----

Для запуска тестов нужно выполнить команду в терминале:
```(shell)
mvn clean test
```

Для сборки проекта нужно выполнить команду в терменали:
```(shell)
mvn clean install
```
